# Global-Youtube-Statistics
The Global YouTube Statistics project analyzes and visualizes the performance of over 1,000 YouTube channels across 50+ countries using Tableau.
It explores global content trends, channel growth, and audience engagement, highlighting top-performing categories and regions.
Interactive dashboards with maps, KPIs, and scatter plots were created to examine relationships between views, subscribers, and revenue.
The project enables data-driven insights for content creators, marketers, and analysts.
It helps identify trending content, top channels, and regional engagement patterns for strategic decision-making.
